Graphs

 

Think of graphs as trees with cycles.  Thus, unlike a tree traversal, you need to make sure you are not moving around in cycles.

A few other diffrences are that trees are typically presumed to have only 2 child nodes, while a node in a graph is typically presumed to have n adjacent nodes. Graphs, unlike trees, can be disconnected. 

 

Basic Questions

Find if there is a path between two given nodes. 

 

Advanced Questions

A Knight is at location (0,0) on a chessboard. Find the minimum number of steps needed to reach square (0,3). 

 

A Knight is at location (0,0) on a chessboard. Enumerate its path (points in needs to visit in order) to (3, 3), if such a path exists. 

Example: The path to (4,2) is (0,0), (2,1), (4,2).
