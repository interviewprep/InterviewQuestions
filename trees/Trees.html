Trees

 

 

A linked list where each node points to two nodes, and no cycles.  The advantage of trees over linked list is that we don't always need to search all the nodes, but can traverse specific paths based on the value you are looking for.

Binary Search Tree: A tree in which left child nodes are less than root, and right child nodes are greater than or equal to root. 

Balanced BST: Height of tree is ~ log n. 

Un-balanaced BST: Height of tree ~ n.

 

Binary Search Tree Time Complexity (balanced, un-balanced)

Insertion: log n,  n

Deletion: log n, n

Search: log n, n
